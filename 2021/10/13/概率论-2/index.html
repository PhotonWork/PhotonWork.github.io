多维随机变量

上一节整理了一维随机变量，包括常见的分布，以及分布函数，概率密度函数相关的概念，这节我们整理一下多维随机变量

一般的，设$X = (X_1,X_2,\dots,X_n) $其中每一个分量，即$X_,\dots,X_n$都是一维随机变量，则称$X$是一个$n$维随机变量

离散型随机变量的分布

这个是$n$维随机变量的概念，我们知道，一维随机变量有概率函数等概念，对于多维随机变量我们也同样有这些概念

定义2.1 以${a_{i1},a_{i2},\dots }$记$X_{i}$的全部可能值($ i=1,2 \dots $),则事件${X_1 = a_{1j_1},X_2=a_{1j_2},\dots ,X_n=a_{1j_n}}$ 的概率$p(j_1,j_2,\dots ,j_n) = P(X_1 = a_{1j_1},X_2=a_{1j_2},\dots ,X_n=a_{1j_n})$,称为随机变量$X = (X_1,X_2,\dots,X_n) $的概率函数或概率分布，概率函数应满足条件$p(j_1,j_2,\dots ,j_n)\geq 0$,$\sum_{j_n}\dots \sum_{j_2}\sum_{j_1} p(j_1,j_2,\dots ,j_n) =1$

离散型$n$维随机变量的一个比较重要的就是多项分布，这个仿照二项分布比较简单这里就不赘述了，本身不是重点

连续型随机变量的分布

定义2.2 若$f(x_1,x_2,\dots,x_n)$是定义在$\mathbb{R}^n$上的非负函数，使得对,$\mathbb{R}^n$中的任何集合$A$，有则称$f$是$X$的(概率)密度函数

同时注意到
这是概率密度函数必须满足的条件

边缘分布

简单来说边缘分布就是只考虑一个变量$x_i$，其他变量先求和，得到的关于$x_i$的函数

这里我们仅仅考虑连续的，以二维为例，假设随机变量$X=(X_1,X_2)$有概率密度函数$f(x_1,x_2)$,此时$X1$和$X_2$的概率密度函数可以这样看

首先考虑$X_1$的概率分布函数$F_1(x_1) = P(X_1\leq x_1)$,它可以写为$P(X_1\leq x_1) = P(X_1\leq x_1,X_2 < \infty)$则

而$\int_{-\infty}^{\infty} f(t_1,t_2)dt_2$是$t_1$的函数，简记为$f_1(t1)$,则原式又可改写为众所周知，概率密度函数就是对概率分布函数求导，两边对于$x_1$求导，得到同理可以得到$X_2$的密度函数为$f_2(x_2) = \int_{-\infty}^{\infty} f(x_1,x_2)_dx_1$

结语及补充

需要注意的是，虽然$X = (X_1,X_2,\dots,X_n)$的分布$F$可以决定任意分量$X_i$的边缘分布，但是反过来却不行。即就算知道了任意分量$X_i$的边缘分布，我们也无法求出$X = (X_1,X_2,\dots,X_n)$的分布

总的$X = (X_1,X_2,\dots,X_n)$的分布不止取决于每个分量的边缘分布，还取决于它们之间的干涉关系，所以我们有时候为了强调$X = (X_1,X_2,\dots,X_n)$的分布是$X_1,X_2,\dots,X_n$作为一个有联系的整体，把$X = (X_1,X_2,\dots,X_n)$称为$X_1,X_2,\dots,X_n$的联合分布
