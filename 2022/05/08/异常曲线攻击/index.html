ејӮеёёжӣІзәҝж”»еҮ»

Created2022-05-08|Updated2022-05-08
|Word count:747|Reading time:3min

ејӮеёёжӣІзәҝжҢҮзҡ„жҳҜйӮЈдәӣзӮ№зҡ„дёӘж•°дёҺжңүйҷҗеҹҹзҡ„йҳ¶ж•°(жӯӨеӨ„еҸӘиҖғиҷ‘зҙ еҹҹ)дёҖж ·зҡ„жӨӯеңҶжӣІзәҝ,еҜ№дәҺиҝҷзұ»жӣІзәҝжҲ‘д»¬жңүдёҖз§Қж”»еҮ»ж–№жі•еҸҜд»ҘеңЁеҫҲеҝ«зҡ„ж—¶й—ҙеҶ…жұӮеҫ—ECDLPгҖӮе…¶дё»иҰҒжҖқи·ҜжҳҜйҖҡиҝҮзҫӨзҡ„еҸҳжҚўпјҢе°ҶжӨӯеңҶжӣІзәҝзҫӨдёҠзҡ„иҝҗз®—еҗҢжһ„жҲҗ$(F_p,+)$пјҢд№ҹе°ұжҳҜжЁЎpзҡ„еҠ жі•зҫӨгҖӮеҜ№дәҺеҠ жі•зҫӨжҲ‘д»¬еҸҜд»ҘеҫҲе®№жҳ“жұӮеҫ—еҜ№еә”еҫ—еҲ°зҡ„ECDLP

дәЁжіҪе°”еј•зҗҶ

иҝҷдәӣиҪ¬жҚўдёӯй—ҙзҡ„зәҪеёҰжҳҜp-иҝӣж•°пјҢе…¶дёӯз”ЁеҲ°зҡ„дёҖдёӘйҮҚиҰҒе·Ҙе…·жҳҜдәЁжіҪе°”еј•зҗҶ(Hensel вҖҳs Lemma)

и®ҫ$f(x)$жҳҜж•ҙзі»ж•°еӨҡйЎ№ејҸ,$k$дёәдёҚе°‘дәҺ2зҡ„ж•ҙж•°пјҢ$p$дёәиҙЁж•°гҖӮиӢҘж•ҙж•°$r$жҳҜдёӢйқўеҗҢдҪҷејҸзҡ„ж №

еҜ№дәҺ

иӢҘ$fвҖҷ(r) \not\equiv 0 \mod p$,еҲҷеӯҳеңЁе”ҜдёҖзҡ„$0 \le t \le p-1$дҪҝеҫ—дёҠејҸжҲҗз«ӢгҖӮ

еҪ“з„¶еҜ№дәҺ$fвҖҷ(r) \equiv 0 \mod p$зҡ„жғ…еҶөиҝҳжңүе…¶д»–зҡ„з»“и®әпјҢиҝҷйҮҢжҲ‘д»¬еҸӘиҖғиҷ‘дёҚзӯүзҡ„жғ…еҶөгҖӮдёҠејҸзҡ„иҜҒжҳҺеҸҜд»Ҙз”Ёжі°еӢ’еұ•ејҖжқҘе®ҢжҲҗ

йҖҡиҝҮдәЁжіҪе°”еј•зҗҶпјҢжҲ‘д»¬еҸҜд»Ҙе®ҢжҲҗдёҖз§ҚliftпјҢеҚіжҠҠе№іж—¶зҡ„жЁЎ$p$зҡ„жғ…еҶөжҸҗеҚҮеҲ°жЁЎ$p^k$зҡ„жғ…еҶөпјҢеҗҢж—¶д№ҹе°Ҷзӣёеә”зҡ„$p$зҡ„е№Ӯзҡ„зі»ж•°з»ҷжҗһеҮәжқҘгҖӮйҖҡиҝҮдәЁжіҪе°”еј•зҗҶжҲ‘д»¬еҸҜд»Ҙе°Ҷ$F_p$дёҠзҡ„жӨӯеңҶжӣІзәҝжҸҗеҚҮеҲ°$Q_p$пјҢд№ҹе°ұжҳҜp-иҝӣж•°зҡ„ж•°еҹҹ

p-иҝӣж•°(p-adic number)

жӯӨеӨ„еҜ№дәҺp-иҝӣж•°дҪңз®ҖиҰҒзҡ„д»Ӣз»Қ

еҜ№дәҺд»»ж„ҸжңүзҗҶж•°$a$д»ҘеҸҠзҙ ж•°$p$,еҸҜеҶҷжҲҗ$a = p^r \frac{m}{n}$е…¶дёӯ$m,n$еқҮдёҺ$p$дә’иҙЁгҖӮз®ҖеҚ•ең°иҜҙе°ұжҳҜе°ҪйҮҸең°жҸҗеҸ–$p$зӣҙеҲ°жҸҗдёҚеҠЁгҖӮ

жҲ‘д»¬е®ҡд№ү

дёә$a$зҡ„йҳ¶гҖӮеҜ№дәҺд»»ж„Ҹе…ғзҙ $a \in Q_p$,еҰӮжһң$ord_p(a) \ge 0$,йӮЈд№Ҳе…¶иў«з§°дёәp-иҝӣж•ҙж•°(p-adic integer)гҖӮжүҖжңүиҝҷж ·зҡ„йӣҶеҗҲз§°дҪң$Z_p$(жіЁж„ҸдёҺ$Z / pZ$иҝҷдёӘе•ҶзҫӨеҢәеҲҶ)

жҲ‘д»¬дёҖиҲ¬е°Ҷpиҝӣж•°иЎЁзӨәжҲҗеҰӮдёӢеҪўејҸ

жӨӯеңҶжӣІзәҝзҡ„жү©еј 

еҜ№дәҺдёҖдёӘе®ҡд№үеңЁ$F_p$дёҠWeierstrassеһӢзҡ„жӨӯеңҶжӣІзәҝ

еҰӮжһңжҲ‘д»¬жғіз”Ёе…¶иЎЁзӨәжңүзҗҶзӮ№пјҢжҲ‘д»¬еҸҜд»Ҙд»Ө$z = -\frac{x}{y},w = -\frac{1}{y}$

йӮЈд№Ҳж–№зЁӢе°ұиҪ¬еҸҳжҲҗ

жҲ‘д»¬еҸҜд»ҘеҜ№е…¶дёҚж–ӯйҖ’еҪ’жұӮеҫ—$w$е…ідәҺ$z$зҡ„иЎЁиҫҫејҸ

е°Ҷе…¶еёҰе…Ҙ$x,y$еҫ—еҲ°
